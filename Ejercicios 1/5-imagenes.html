HTML 5

El 28 de octubre de 2014 se aprobó la Recomendación HTML 5, culminando así diez años de trabajo. en 2004 se formo WHATWG, un grupo informal de individuos y
empresas decididos a ipulsar el desarrollo de HTML, que se enontraba bloqueado en el W3C desde el año 2000
En 2007 el W3C retomó el trabajo sobre HTML y siete años despues se alcanzó el objetivo
Pero el HTML 5 no era más que un primer paso. El 1 de noviembre de 2016 se aprovó la recomendación HTML 5.1, ampliando y clarificando el HTML 5, y el W3C ya está
trabajando en el HTML 5.2, previsto para finales de 2017
